## Especificaciones técnicas para filmar anuncios que CONVIERTEN

---

## Especificaciones técnicas

| Elemento | Requisito |
|----------|-----------|
| **Resolución** | 1080×1920 píxeles (9:16 vertical) |
| **Framerate** | 30 fps (60 fps si hay cámara lenta) |
| **Formato de exportación** | MP4 H.264 |
| **Bitrate** | Mínimo 8 Mbps |
| **Duración** | 15s (orgánico) / 30s (ads pagados) |
| **Subtítulos** | Grandes, blanco (#FFFFFF) + sombra negra (#000000 60% opacity) |
| **Máx líneas de texto** | 2 líneas por frame |
| **Fuente de texto** | Montserrat Bold o Impact — legible en móvil |

---

## Reglas de iluminación

| Situación | Configuración |
|-----------|--------------|
| **Luz natural** | Ventana grande, luz indirecta, horario 10am–2pm |
| **Luz artificial** | Softbox o anillo de luz a 45°, temperatura 5000K–5600K |
| **Nunca usar** | Luz amarilla cálida (<4000K) — hace look barato |
| **Contraste** | El pelo debe notarse — usa fondo oscuro si pelaje es claro y viceversa |

## Dirección de arte

### Qué SÍ hacer

- ✅ Hogar real — sofá usado, almohadas, mantas
- ✅ Mascota real caminando o acostada cerca
- ✅ Manos reales (con anillos, uñas, textura de piel)
- ✅ Desorden controlado (parece auténtico pero no descuidado)
- ✅ Ropa casual (nada formal ni de oficina)

### Qué NO hacer jamás

- ❌ Fondos blancos de estudio
- ❌ Producto sobre fondo degradado
- ❌ Manos con guantes o demasiado perfectas
- ❌ Música genérica de biblioteca sin ritmo
- ❌ Textos pequeños o difíciles de leer en móvil

## Ritmo de edición

| Sección | Ritmo de corte | Efecto |
|---------|---------------|--------|
| Hook (0–3s) | Cada 0.5–1s | Mantiene atención |
| Problema (3–10s) | Cada 2–3s | Construye tensión |
| Solución (10–20s) | Cada 3–5s | Muestra proceso real |
| Resultado (20–25s) | Cámara lenta (0.5x) | Dramatiza el logro |
| CTA (25–30s) | 1 plano fijo de 5s | Precio legible |

---

## Sonido

| Elemento | Especificación |
|----------|---------------|
| **Música** | Trending TikTok o librería libre (Epidemic Sound, Uppbeat) |
| **Género** | Upbeat, energía positiva, ritmo marcado |
| **FX gancho** | Whoosh, scratch, risa, sorpresa |
| **FX solución** | Satisfacción — sonido de limpieza, suspiro de alivio |
| **Volumen voz** | -6dB a -3dB (sobre música a -20dB) |

## Checklist pre-filmación

- [ ] Sofá/tela oscura para contraste con pelo claro
- [ ] Pelo de mascota real (no simulado) para demostración
- [ ] Mascota presente (al menos en 1 plano)
- [ ] FurSweep™ listo + limpio
- [ ] Rodillo desechable viejo para comparación
- [ ] Trípode o estabilizador
- [ ] Anillo de luz o softbox
- [ ] Microphone de solapa (si hay voz)
- [ ] Script impreso para el talento
- [ ] Agua + despejar el set

---

## Checklist post-producción

- [ ] Recortar a exactamente 15s o 30s
- [ ] Subtítulos grandes — verificar en móvil
- [ ] Color grading: contraste alto, saturación +10%
- [ ] Frame final: precio visible mínimo 3s
- [ ] Flecha CTA animada o estática grande
- [ ] Exportar en MP4 H.264 1080x1920
- [ ] Subir a plataforma y verificar vista previa móvil

## Si el anuncio no funciona — diagnóstico rápido

| Síntoma | Causa probable | Solución |
|---------|---------------|----------|
| Baja retención en 0–3s | Hook débil | Cambiar gancho, más contraste visual |
| Muchos views, pocos clics | CTA no claro | Precio más grande, flecha más visible |
| Bajas conversiones | Oferta no urgente | Agregar escasez o límite de tiempo |
| Muchos clics, pocas compras | Precio alto o página lenta | Revisar landing page + oferta |
