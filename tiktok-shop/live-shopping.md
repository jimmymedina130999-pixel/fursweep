## Estructura de un Live de 30 minutos

### Minuto 0-5: Enganche
| Acción | Diálogo sugerido |
|--------|-----------------|
| Mostrar producto en mano | "Bienvenidos a mi Live. Hoy les traigo el producto que está revolucionando la limpieza del hogar: FurSweep™" |
| Hacer primera demostración | "Mira esto... pasas UNA vez y mira todo el pelo que saca" |
| Interactuar | "¿Cuántos de ustedes tienen perro o gato? Pongan 🐶 o 🐱 en los comentarios" |

### Minuto 5-15: Demostración principal
| Acción | Diálogo sugerido |
|--------|-----------------|
| Demostración en sofá oscuro | "Mira la diferencia. Este sofá estaba 'limpio'... o eso creía" |
| Mostrar pelo acumulado | "Esto es lo que tu sofá tiene AHORA y no sabías" |
| Demostración en ropa | "¿Ven? También funciona en ropa. Adiós a los rodillos desechables" |
| Comparación vs rodillo | "Mira la diferencia. El rodillo se satura, FurSweep™ sigue funcionando" |

### Minuto 15-20: Ofertas exclusivas Live
| Acción | Diálogo sugerido |
|--------|-----------------|
| Anunciar descuento live | "SOLO para los que están en este Live: envío gratis + 50% OFF" |
| Mostrar packs | "Pack de 2 es el más vendido. Pero si compras el de 3, el ahorro es aún mayor" |
| Urgencia | "Tengo solo [X] unidades disponibles con este descuento. Cuando se acaben, se acaban" |

### Minuto 20-25: Preguntas y respuestas
| Pregunta frecuente | Respuesta |
|--------------------|-----------|
| "¿Se puede lavar?" | "Agua y jabón, 30 segundos y queda como nuevo" |
| "¿Funciona en gatos?" | "Para todo tipo de pelo: perros, gatos, conejos, cualquier mascota" |
| "¿Daña la tela?" | "No, es 100% seguro. Lo puedes usar en cualquier tela" |
| "¿Cuánto tarda en llegar?" | "3-7 días hábiles. Envío gratis" |

### Minuto 25-30: Cierre con urgencia
| Acción | Diálogo sugerido |
|--------|-----------------|
| Resumen | "Últimos minutos. 50% OFF + envío gratis. Garantía de 30 días" |
| CTA final | "Compra ahora directamente desde la shop. Link en el carrito 👇" |
| Agradecimiento | "Gracias por acompañarme. Sígueme para más lives y ofertas" |

## Preparación pre-Live

| Elemento | Checklist |
|----------|-----------|
| Producto | Tener 2-3 FurSweep™ limpios + superficies para demostrar |
| Vestimenta | Ropa oscura para que se vea mejor el pelo |
| Iluminación | Luz frontal cálida + luz de relleno |
| Audio | Micrófono de solapa o micrófono externo |
| Notificaciones | Configurar aviso de Live 1 hora antes |
| TikTok Shop | Verificar que los productos estén activos y con stock |

## Estrategia de ofertas para Live

| Tipo de oferta | Condición | Duración |
|----------------|-----------|---------|
| **Envío gratis** | Todos los pedidos durante el Live | Durante el Live |
| **Descuento Live** | 55% OFF en lugar de 50% | Solo 10 primeras órdenes |
| **Pack especial** | 4 unidades al precio de 3 | Exclusivo Live |
| **Regalo sorpresa** | FurSweep™ mini de regalo | Primeros 20 compradores |

## Métricas a medir post-Live

| Métrica | Objetivo |
|---------|----------|
| Espectadores máximos | >50 concurrentes |
| Duración promedio de visualización | >5 minutos |
| Tasa de conversión del Live | >3% de espectadores → compra |
| Ingresos por Live | >$100 |
| Nuevos seguidores | >20 por Live |
