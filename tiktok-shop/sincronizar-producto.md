# 🔄 Guía Rápida: Sincronizar FurSweep™ a TikTok Shop
## Usando TikTok Shop Connector (CedCommerce)

---

## Paso 1: Verificar que el producto está sincronizado

1. Ve a **Admin Shopify** → **Apps** → **TikTok Shop Connector (CedCommerce)**
2. Panel izquierdo → **Productos** → **Lista de productos**
3. Busca **FurSweep™ Pro**
4. ¿Ves el producto con estado **"Sincronizado"** o **"Pendiente"**?

| Estado | Qué hacer |
|--------|-----------|
| ✅ **Sincronizado** | Ve al paso 2 |
| ⏳ **Pendiente** | Haz clic en el producto → **Sincronizar ahora** |
| ❌ **No aparece** | Ve a **Agregar producto** → busca "FurSweep-Pro" → selecciona → **Sincronizar** |

## Paso 2: Sincronizar manualmente (si es necesario)

Si el producto aparece como "Pendiente" o "Error":

1. Selecciona **FurSweep™ Pro** de la lista
2. Haz clic en **Editar / Sync Now**
3. Verifica que estos campos estén correctos:

| Campo | Valor requerido |
|-------|----------------|
| **Título** | FurSweep™ - Pet Hair Remover Reusable 50% OFF |
| **Precio** | $24.99 / $39.99 / $49.99 |
| **Inventario** | > 100 unidades cada variante |
| **Categoría** | Home & Garden > Cleaning Tools |
| **Condición** | Nuevo |
| **Peso** | 100g / 200g / 300g |
| **Dimensiones** | 15 x 8 x 3 cm |

4. Haz clic en **Sincronizar**
5. Espera 2-5 minutos
6. Refresca la página → debe aparecer **"Published"** o **"Active"**
