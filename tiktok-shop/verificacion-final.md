## Lo que debes hacer TÚ (5 minutos)

### Paso 1: Sincronizar producto
1. Ve a **Shopify Admin** → **Apps** → **TikTok Shop Connector (CedCommerce)**
2. **Productos** → busca **FurSweep™ Pro**
3. Si no aparece → **Agregar producto** → selecciónalo → **Sincronizar**
4. Espera 2-5 min → estado debe cambiar a **Published** / **Active**

### Paso 2: Copiar descripción optimizada
Usa la descripción de `sincronizar-producto.md` (ya está en inglés para mercado US)

### Paso 3: Verificar en TikTok Shop
1. Ve a **seller.tiktok.com** → cuenta **Flything**
2. **Products** → **All Products**
3. Busca FurSweep™ → estado debe ser **Active**
4. Haz clic en **Preview** → prueba que se vea bien

### Paso 4: Obtener Pixel ID
1. Ve a **ads.tiktok.com** → **Events** → **Manage events**
2. Copia el **Pixel ID**
3. Pásamelo → lo actualizo en la landing page
